mglSelectFeatures(map, selectingFeatures, layers, field, filter) → {Number}

This function takes a polygon or point and selects other features in the map that intersect this feature. It returns the features as well as an array contaiing a list of unique ids for these features based on the properties field provided.
Parameters:
Name Type Description
map object Mapbox map object
selectingFeatures object GeoJSON FeatureCollection
layers Array Array of Mapbox layers to query
field string Field used to filter the features
filter function Function used on each feature to optionally filter out features.
